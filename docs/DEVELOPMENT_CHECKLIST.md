1. **開發階段**: 按照新功能開發檢查清單執行
2. **測試階段**: 使用測試檢查清單驗證
3. **部署階段**: 按照部署檢查清單操作
4. **維護階段**: 使用維護檢查清單定期檢查

### 自定義檢查清單
- 根據專案需求調整檢查項目
- 添加專案特定的檢查點
- 定期更新檢查清單內容
